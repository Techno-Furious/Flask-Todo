About This WebApp

Overview

This TODO application allows you to:

  • Add new tasks with titles and descriptions.
  • Update existing tasks to keep track of changes.
  • Delete tasks when they are no longer needed.
  • Search for specific tasks to quickly find what you need.

Motivation

As a student, keeping track of assignments, deadlines, and personal tasks can be challenging. I created this app to make it easier to stay organized and focused on what needs to be done.

I used

  • Flask for the web framework
  • SQLite for the database
  • Bootstrap for styling
  • HTML and CSS for structure and design
